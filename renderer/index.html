Backtest

Replay history through the exact strategy code the live engine runs
Backtests flatter: perfect data, instant fills, no competition for shares. Use them to compare strategies and parameters — not to predict profits. The simulation is portfolio-style: your max-open-positions cap and daily loss limit apply across all symbols together, entries fill at the next bar's open, stops win ties, strategy exits fill at the bar's close, and slippage is charged both ways.

Settings

Trading Mode

Paper mode uses Alpaca's simulator — real market data, fake money.
Each mode keeps its own strategy, engine, risk, and stock settings. Switching modes stops the engine.

Alpaca API Keys

Get free keys at alpaca.markets ↗

Paper keys

Live keys

Keys are stored on this computer only, encrypted with Windows account encryption (DPAPI).

Engine

The schedule auto-starts and auto-stops the engine each weekday (Eastern Time). The app must be running — turn on tray mode under Application so it keeps trading with the window closed. If you stop the engine manually mid-window, it stays stopped until the next day.

Risk Controls

Every buy is placed as a bracket order — its stop-loss and take-profit live at the broker, so positions stay protected even if this app closes.

Application

With tray mode on, closing the window keeps the engine trading in the background — reopen or quit from the tray icon next to your clock.

Updates

Updates are delivered automatically from this app's GitHub releases — no setup needed. The feed URL above is an optional override if you ever want to self-host updates on a different HTTPS folder.

Email Reports

Gmail users: create an app password (Google Account → Security → 2-Step Verification → App passwords) and use it here instead of your normal password.

Data

Export/import moves your configuration between computers — API keys never travel and imports always land in paper mode. The support bundle packages recent technical logs into one text file for troubleshooting.

Reset returns every setting to its recommended default. Your API keys and trade history are kept. Technical logs cover engine activity, broker errors, and crashes for troubleshooting; they are deleted automatically after 7 days.

Welcome to TradeForge 🚀

TradeForge trades stocks automatically through your own Alpaca brokerage account using classic day-trading strategies with strict, broker-side risk protection.

  • You'll start in paper mode — real market data, simulated money.
  • First step: a free Alpaca account (takes a few minutes).
  • Once it's created, generate paper trading API keys from the Alpaca dashboard.

Connect your paper account

Paste the paper-trading keys from your Alpaca dashboard. They're stored only on this computer, encrypted with Windows account encryption.

Pick a starting strategy

One strategy runs at a time. Each has an ⓘ explaining exactly how it trades — and you can compare all ten on historical data in the Backtest tab before committing.

You're set — the honest checklist

  • Backtest first. Run "Compare all 10 strategies" and let data pick your starting point.
  • Paper trade for weeks. The "Am I ready for live?" report in Settings scores you honestly.
  • The daily loss limit is your friend. It's pre-set to $300 — keep it tight.
  • Press ▶ Start Engine on the Dashboard during market hours to begin.
